## Firebase Email Registration & Login App

This project is a simple web application built using **HTML**, **Tailwind CSS**, and **Firebase**.

### Features
- User Registration with Email & Password
- Login with Firebase Authentication
- Store user details (username, phone) in Firebase Realtime Database
- Auto-detect logged-in state

### Files
- `index.html` - Registration page
- `login.html` - Login page

### Setup
1. Clone the repository
2. Open `index.html` in your browser
3. Register or Login using your email and password
