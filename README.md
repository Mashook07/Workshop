## Firebase Email Registration & Login App

A complete web application built using **HTML**, **Tailwind CSS**, and **Firebase**.

### Features
- User Registration with Email & Password
- Login with Firebase Authentication
- Password Reset via Email
- User Dashboard with profile details
- Store user details in Firebase Realtime Database
- Auto-detect logged-in state and redirect

### Pages
- `index.html` - Registration page
- `login.html` - Login page
- `reset-password.html` - Password reset page
- `dashboard.html` - User dashboard

### Tech Stack
- HTML5 + Tailwind CSS
- Firebase Authentication (v9)
- Firebase Realtime Database (v9)

### Setup
1. Clone the repository
2. Replace `firebaseConfig` in each file with your own Firebase project config
3. Open `index.html` in your browser to get started
